A web-based Smart Grid dashboard built with Python and deployed on Streamlit Cloud, designed for real-time monitoring, anomaly detection, and data quality assessment.

The system simulates live sensor data and provides interactive visualizations for voltage, current, frequency, and power metrics. It integrates both classical statistical techniques and machine learning models (Isolation Forest, Local Outlier Factor, One-Class SVM, and PCA reconstruction) to detect anomalies and validate system behavior.

The project introduces a modular validation and quality assessment framework, evaluating data based on completeness, stability, and accuracy. The architecture is designed to be adaptable to multiple data sources, including CSV files, APIs, and industrial systems (e.g., SCADA/PLC).

 Live demo link: https://smart-dashboard-tematwebtest.streamlit.app
